## 简单的社交网站签到脚本

### Usage
* V2EX
```python
v2ex = v2ex('your username', 'your password')
v2ex.login()
v2ex.sign()
```
* 优酷
```python
youku = youku('your cookie')
youku.sign()
```

* 网易云音乐
```python
neteasemusic = neteasemusic('your cookie')
neteasemusic.sign()
```

* AcFun
```python
acfun = acfun('your username', 'your password')
acfun.login()
acfun.sign()
```
